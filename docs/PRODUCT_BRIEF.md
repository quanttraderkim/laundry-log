# Product Brief

## A. 목적

사용자가 옷을 언제 빨았는지 기억하지 않아도 되도록, 모바일 웹에서 마지막 세탁일과 착용 횟수를 빠르게 기록하고 확인하는 서비스입니다.

## B. 핵심 판단

| 항목 | 판단 |
|---|---|
| 1차 경험 | 옷 목록에서 상태를 보고 `오늘 입음`, `세탁 완료`를 바로 기록 |
| 우선 플랫폼 | 설치 없는 모바일 웹 |
| 초기 사용자 | 혼자 사는 사용자, 운동복/출근복을 자주 돌려 입는 사용자 |
| MVP 목표 | 기록 부담을 줄이고 세탁 필요 여부를 한 화면에서 판단 |

## C. MVP 범위

| 기능 | 설명 | 우선순위 |
|---|---|---|
| 옷 등록 | 이름과 종류만 입력 | P0 |
| 마지막 세탁일 표시 | 며칠 전 빨았는지 표시 | P0 |
| 세탁 완료 기록 | 오늘 날짜로 갱신하고 착용 횟수 초기화 | P0 |
| 착용 기록 | 착용 횟수를 늘려 세탁 권장 판단에 반영 | P0 |
| 상태 라벨 | 깨끗함, 곧 세탁, 세탁 권장으로 구분 | P0 |
| 계정/동기화 | DB 도입 후 구현 | P1 |

## D. 보류 범위

세탁기 연동, 의류 이미지 인식, 상세 세탁 코스 추천, 가족 공유는 MVP 이후로 둡니다. PRD에서 중요도가 높게 나오면 P1로 재평가합니다.
